# TiebaBoost

TiebaBoost是一款本着提升贴吧使用体验而开发的chrome插件。

大家有兴趣的话，下来玩玩，顺便帮我测试一下，看看有没有bug。

如果大家反馈没有bug，或者修复大家反馈的bug后，下一步打算做：
1、关注某个主题贴，若有回复则提醒；（战史撕比、吊打瘪杂、各类讨论贴）
2、关注某个楼主在某个主题贴的发言，若有回复则提醒；（直播贴）
3、关注某个人，在任何一个地方发言都有提醒；（追星功能）
